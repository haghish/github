echo example

This is the example created for the manuscript entitled “GitHub: A Better Place for Developing, Maintaining, and Hosting Statistical Software”. The example shows that how markdoc package can be used to create a Stata help file from Markdown and how the github package package can be used to create Stata package installation files.

Stata help files

To create the echo.sthlp help file, navigate to the current working directory and then run:

     . markdoc echo.ado, mini export(sthlp) replace
